## Debug mode run

CTRL + shift + d

## (Re)-RUN

CTRL + r

## Step Into

F7

## Step Over

F8

## Step Out

Shift + F8

## Go to Class

CMD + o

## Find

CMD + F

## Find in path

CMD + Shift + F

## Go to declaration

CMD + b
